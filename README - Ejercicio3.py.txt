Descripción
Este programa define una clase `Estudiante` que representa a un estudiante con atributos privados para el nombre, la edad y una lista de notas. La clase incluye métodos para agregar notas, calcular el promedio de notas y obtener la información del estudiante.

Estructura del Código
El código contiene lo siguiente:
  - Clase `Estudiante`**:
  - `__init__(self, nombre, edad)`: Constructor que inicializa los atributos del estudiante.
  - `agregar_nota(self, nota)`: Método que agrega una nota a la lista, validando que esté entre 0 y 100.
  - `calcular_promedio(self)`: Método que calcula y devuelve el promedio de las notas.
  - `obtener_nombre(self)`: Método que devuelve el nombre del estudiante.
  - `obtener_edad(self)`: Método que devuelve la edad del estudiante.

Ejemplo de uso:
  - Se crea un estudiante "Kevin Tapia" con edad 20.
  - Se agregan tres notas: 85, 90 y 78.
  - Se muestra el nombre y la edad del estudiante.
  - Se calcula y muestra el promedio de notas.
