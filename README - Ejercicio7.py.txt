Descripción
Este programa define una clase `TarjetaCredito` con un método estático para validar números de tarjetas de crédito usando el algoritmo de Luhn.

Estructura del Código
El código contiene lo siguiente:
  - Clase `TarjetaCredito`:
  - `validar_tarjeta(numero)`: Método estático que valida un número de tarjeta utilizando el algoritmo de Luhn. Se asegura de que el número contenga solo dígitos y aplica la lógica de validación.

Ejemplo de uso:
  - Se proporciona un número de tarjeta de crédito.
  - Se valida el número de tarjeta usando el método estático.
  - Se imprime si el número de tarjeta es válido o no.
